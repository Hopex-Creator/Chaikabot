## Instalar
Siga los pasos a continuación!

```bash
> termux-setup-storage
(después de eso toca el permiso)
> pkg install git
> pkg install ffmpeg
> pkg install wget
> pkg install nodejs
> pkg install npm
> git clone https://github.com/Hopex-Creator/Chaikabot.git
> cd ToiniuBot
> bash install.sh
```

## Usar

```bash
> npm start
```
